# Markdown Syntax

## Heading levels in Markdown

| Markdown Syntax | Output |
|:-----------------:|:-----------------:|:----------------:|
|#|# Heading Level 1|
|##|## Heading Level 2|
|###|### Heading Level 3|

## Emphasis in Markdown

| Markdown Syntax | Content | Output |
|:-----------------:|:-----------------:|:----------------:|
|Single Asterisk|Harsh|*Harsh*|
|Double Asterisk|Harsh|**Harsh**|
|Stikethrough|Harsh|~Harsh~|
